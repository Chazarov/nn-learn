# topologic distance 

Calculate the raw distance between the positions of the neurons on the map


## Евклидово топологическое расстояние

**Формула:**
$$d_i = (i - c)^2$$

**Где:**
- $d_i$ — топологическое расстояние до нейрона $i$
- $i$ — индекс нейрона на линейной карте $(0, 1, ..., N-1)$
- $c$ — индекс нейрона-победителя (winner_idx)

## Манхэттенское топологическое расстояние

**Формула:**
$$d_i = |i - c|$$

**Где:**
- $d_i$ — топологическое расстояние до нейрона $i$
- $i$ — индекс нейрона на линейной карте $(0, 1, ..., N-1)$
- $c$ — индекс нейрона-победителя (winner_idx)
