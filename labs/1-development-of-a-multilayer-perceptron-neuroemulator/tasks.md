# Реализовать фронтенд

проект в темной теме на React. В стиле пиксельной аркадной игры. Черно белый (фон черный, шрифт белый). Можно с приятными мягкими отенками 
для подражания нейлоновым лампам. Все элементы интерфейса на английском.

1) окно входа: пароль, адрес электронной почты и ссылка "уже есть аккаунт" на английском.
И подпись над адресом: Друг, я не буду отправлять тебе ничего на почту. Она нужна чтобы ты не потерял доступ к своей работе на моем сервисе.

2) Элемент - иконка знак вопроса при нажатии по которой должна открываться ссылка , которую я смогу указать в ручную. 

3) Главная страница -  хеадер с надписью "эмулятор нейронной сети" и справа сверху  маленькая иконка вопроса 
слева на странице - список проектов пользователя. В конце списка знак + для добавления нового проекта. Около каждого 
элемента списка - иконка крестик . При нажатии на нее проект удаляется
Под списком проектов - список файлов csv с обучающими примерами. Рядом с каждым элементом этого списка - иконка Х (крестик), при нажатии на который файл удаляется

4) При нажатии на кнопку "добавить проект" -  по центру появляется небольшой интерфейс для указания количества и размера скрытых слоев перцептрона. сделай так чтобы максимум их могло быть 10 а нейронов в слое 20. При этом каждому слою отдельно можно задать свое количество нейронов. В правй части страницы  - выпадающий список с выбором функции активации  "RELLU" или "SIGMOID" .  а также выпадающий список с выбором файла csv который будет использован для создания перцептрона
 Под этим списком две большие кнопки "СОЗДАТЬ" . Когда проект создан (была нажата кнопка создать и получен ответ от сервера с запросом на создание этого перептрона) чуть ниже добавляется такая же кнопка "ОБУЧИТЬ". При нажатии на эту вторую кнопку отправляется запрос на обучение перцептрона. Вне зависимости от того какое действие из этих двух выполнилось выполнилось . Сразу после него выполняется запрос get_image который получает изображение перцептрона и выводит его в центр экрана вместо интерфейса для ввода значений размерности скрытых слоев.

5) когда Проект создан и изображение перцептрона выведено в центр экрана то под ним должен появится интерфейс для ввода входного вектора для тестирования перцептрона. Размерность этого вектора ты должен получить из данных которые  приходят как результаты из методов /get_answer и /learn. project["nn_data"]["input_size"]. и под ним большую кнопку "GetAnswer".
При нажатии ты должен получать ответ на запрос get_answer и все его содержимое выводить в поле, находящееся в самом низу под этой кнопкой. Это поле - просто квадрат в котором должен быть выведен отформатированый json ответ запроса 

6) Когда пользователь нажимает на какой то из уже существующих проектов, то весь интерфейс должен подстраиваться соответственно под этот проект. 
7) Функцию активации можно поменять в любой момент но строго  после создания нейросети.
