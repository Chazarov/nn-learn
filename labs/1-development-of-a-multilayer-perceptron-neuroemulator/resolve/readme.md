# Учебный эмулятор многослойного перцептрона


Этот проект создан с целью изучения работы основного компонента нейронной сети - многослойного перцептрона (полносвязной нейронной сети). В рамках подготовки к реализации этого проекта и во время его реализации я пользовался различными источниками и оставил их в проекте в качестве подсказок. Это уже переработанная мной лично информация , которая помогает легче понять код этого проекта и тему которой он посвящен. 


Вы можете изучить фундаментальные процессы ,из которых состоит любая нейронная сеть попробовав сервис = эмулятор доступный по адресу https://nnemulator.science.zaharow.ru. И одновременно проанализировав его код, представленный в этом репозитории с помошью оставленных мною подсказок и документации. Приятного изучения;) **Chazarow**

**В этом проекте вы найдете:**
1) написанный вручную с достаточно подробными комментариями модуль расчта для процессов полносвязной нейронной сети и ее поведения. Включая обучение, forwardpropagation, backpropagation, функции активации и тд. Все это с справочной информацие вы найдете [тут](https://github.com/Chazarov/nn-learn/tree/master/labs/1-development-of-a-multilayer-perceptron-neuroemulator/resolve/simple_emulator/nn_logic) . Модули написаны с использованием только базовых библиотек python без использования numpy и других для максимально быстрого их понимания.
2) Полностью самодельный, очень наглядный модуль визуализации состояния нейронной сети. О нем более подробно можно узнать [тут](https://github.com/Chazarov/nn-learn/tree/master/labs/1-development-of-a-multilayer-perceptron-neuroemulator/resolve/simple_emulator/nn_logic/visualisation).
3) Архитектуру для многопользовательской систем
4) frontend-react приложение с удобным пользовательским интерфейсом , чтобы поигрваться с эмулятором. Удобно и наглядно изучить влияние различных параметров на функциональность нейронной сети.
5) Настроенную конфигурацию docker-compose для разворачивания системы на сервере
