# нейроэмулятор многослойного перцептрона

## Архитектура

### нейроэмулятор

**Инструменты**:
- python
    - pydantic
    - numpy

**архитектура**
Класс многослойного перцептрона. Должен иметь следующие возможности:
1) Сохранение весов в файл
2) Загрузка весов из файла
3) Хранение размеров слоев
4) Изменение размера слоев
5) Список матриц весов

6) Изменение функции активации для всего перцептрона


### визуализация

**идея** 
подразумевает возможно огромное количество нейронов в будующем. Поэтому необходимо реализовать максимально оптимизированный алгоритм рисования который будет делать сразу снимок текущего состояния нейросети и как то визуализировать его исходя из промежуточных данных.
