# Учебный эмулятор многослойного перцептрона

Этот проект создан с целью изучения работы основного компонента нейронной сети — многослойного перцептрона (полносвязной нейронной сети). В рамках подготовки к реализации этого проекта и во время его реализации я пользовался различными источниками и оставил их в проекте в качестве подсказок. Это уже переработанная мной лично информация, которая помогает легче понять код этого проекта и тему, которой он посвящен.

Вы можете изучить фундаментальные процессы, из которых состоит любая нейронная сеть, попробовав сервис — эмулятор, доступный по адресу [https://nnemulator.science.zaharow.ru](https://nnemulator.science.zaharow.ru). Поможет вам в этом код этого эмулятора, оставленные мною подсказки и документация. Приятного изучения;) **Chazarow**

**В этом проекте вы найдете:**
1) написанный вручную с достаточно подробными комментариями модуль расчета для процессов полносвязной нейронной сети и ее поведения, включая обучение, forward propagation, back propagation, функции активации и т.д. Все это со справочной информацией вы найдете [тут](https://github.com/Chazarov/nn-learn/tree/master/labs/1-development-of-a-multilayer-perceptron-neuroemulator/resolve/simple_emulator/nn_logic). Модули написаны с использованием только базовых библиотек Python без использования NumPy и других для максимально быстрого их понимания.
2) полностью самодельный, очень наглядный модуль визуализации состояния нейронной сети. О нем более подробно можно узнать [тут](https://github.com/Chazarov/nn-learn/tree/master/labs/1-development-of-a-multilayer-perceptron-neuroemulator/resolve/simple_emulator/nn_logic/visualisation).
3) архитектуру для многопользовательской системы.
4) frontend-React-приложение с удобным пользовательским интерфейсом, чтобы поиграться с эмулятором. Удобно и наглядно изучить влияние различных параметров на функциональность нейронной сети.
5) настроенную конфигурацию docker-compose для развертывания системы на сервере.
