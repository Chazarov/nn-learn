# Самоорганизовывающаяся сеть (карта) кохонена(англ. Self‑Organizing Map, SOM)

Это разновидность нейронных сетей с обучением без учителя (неконтролируемым обучением). Идея предложена финским учёным Теуво Кохоненом в начале 1980‑х годов.

**Ключевые особенности:**

- Обучение без учителя. В процессе обучения сеть анализирует только входные данные — без сравнения выходов нейронов с эталонными значениями. Она «учится» понимать структуру данных.
- Топологическая организация. Расположение нейронов в сети отражает сходство между входными данными: похожие примеры группируются рядом на карте, а разные — располагаются дальше друг от друга.
- Проецирование многомерных данных в пространство меньшей размерности (чаще всего — в двумерное). Это позволяет визуализировать сложные данные в удобной форме.
- Соревновательное обучение. В каждом шаге обучения выбирается «нейрон‑победитель» (Best Matching Unit, BMU) — тот, чьи веса наиболее близки к текущему входному вектору. Затем веса этого нейрона и его соседей подстраиваются под входные данные.

**Структура сети:**

- Два слоя: входной (принимает данные) и выходной (самоорганизующаяся карта).
- **!!!Выходной слой состоит из нейронов (узлов), расположенных в виде сетки (обычно двумерной — прямоугольной или шестиугольной)!!!!!**.
- Важно понимать, что функция топологического расстояния никак не относится к самой топологии. Топология сети - это изначальный ее параметр. А функция топологического расстояния необходима только на этапе обучения для понимания степени влияния на конкретный нейрон. 
- Каждый нейрон описывается:
   - вектором весов m (той же размерности, что и входные данные);
   - координатами r на карте.



# Использование
Основные задачи:

- Кластеризация (классификация без учителя). Разбиение данных на группы (кластеры) схожих объектов. Например, группировка клиентов по покупательскому поведению.
- Визуализация многомерных данных. Преобразование данных высокой размерности в двумерную карту, где близкие точки соответствуют схожим объектам. Это помогает выявить скрытые закономерности.
- Разведочный анализ данных. Изучение структуры данных перед построением более сложных моделей. Карты Кохонена помогают понять, есть ли в данных естественные группы, выбросы и т.д.
- Обнаружение аномалий. Если новый объект не попадает ни в один из известных кластеров, он может быть признан аномалией.
- Сжатие данных и выделение признаков. Уменьшение размерности данных с сохранением топологической структуры.
- Прогнозирование. На основе кластеризации можно предсказывать поведение объектов (например, лояльность клиентов).
- Квантизация цветов. В компьютерной графике — сокращение числа цветов в палитре с сохранением визуальной схожести.
- Анализ временных рядов и сигналов. Выявление повторяющихся паттернов в данных (например, в финансовых временных рядах или медицинских сигналах).

# Методы визуализации сетей кохонена. 
Способность визуализации это ключевая особенность и преимущество сетей кохонена , поэтому о них необходимо поговорить отдельно.

## 1) Раскраска карт по компонентам
Этот метод предполагает создание нескольких карт, каждая из которых соответствует одному параметру (компоненте) входных данных. Ячейки карты раскрашиваются в разные цвета или оттенки в зависимости от значения весов нейронов, соответствующих этому параметру. Например, можно использовать градации серого: ячейки с минимальными значениями компонента будут чёрными, а с максимальными — белыми. Также применяются цветовые градиенты.

Полученный набор раскрасок образует «атлас», который отображает расположение компонент, связи между ними и относительное расположение различных значений. Это позволяет анализировать, по каким конкретно параметрам проявляется сходство объектов.


- Количество карт в атласе = количество параметров  (признаков входных данных)
- Количество ячеек в одной карте берется из размерности сети если в сети 5 и 6 неронов (входной и выходной слой), то карта будет размером 5*6 = 30 ячеек
- В каждой клетке — вес связи между параметром и конкретным нейроном карты Кохонена.
- Цвет ячейки определяется значением веса для определенного параметра у этого нейрона после обучения сети

## 2) Унифицированная матрица расстояний (U-matrix)
U-matrix используется для визуализации расстояний между векторами весов нейронов и их ближайшими соседями. Для каждого нейрона вычисляется расстояние до его соседей, и эти значения используются для определения цвета узла на карте. Обычно применяют градации серого: чем больше расстояние, тем темнее отрисовывается узел. Узлам с наибольшим расстоянием между ними и соседями соответствует чёрный цвет, а близлежащим узлам — белый.

- U-matrix помогает определить границы между кластерами: светлые области позволяют объединить нейроны в кластеры, а тёмные — провести границы между ними
- U‑matrix даёт одну карту (не набор карт).
- Размер карты совпадает с размером карты Кохонена (число нейронов выходного слоя).
- В каждой ячейке — среднее расстояние между весами данного нейрона и его соседей.
- Цвет показывает степень различия между нейроном и его окружением: тёмные зоны = границы кластеров, светлые зоны = плотные кластеры.

## 3) Карты выходов нейронов 
показывают модель взаимного расположения входных примеров. Очерченные области на карте представляют собой кластеры, состоящие из нейронов со схожими значениями выходов. 

- Размер карты выходов нейронов = размер карты Кохонена, то есть заданный размер сетки выходного слоя (например, 10×10 или 5×8). Число ячеек = число нейронов выходного слоя.
- Количество карт выходов в атласе = 1. Карта выходов — это одна двумерная сетка, отражающая итоговое расположение нейронов и группировку входных данных.
- За что отвечает каждый «атом» (ячейка) карты = представляет один нейрон выходного слоя карты Кохонена. Ячейка показывает, какие входные векторы «привязаны» к этому нейрону (то есть какие примеры из данных оказались ближе всего к его вектору весов).

U‑matrix (унифицированная матрица расстояний)
Что в ячейке: числовое значение — среднее расстояние между вектором весов данного нейрона и векторами весов его ближайших соседей (обычно евклидово расстояние).

Цель: показать границы между кластерами. Помогает понять, где одни группы нейронов (кластеры) отделяются от других.

Как визуализируют: одну карту, где:

тёмный цвет (чёрный/тёмно‑серый) → большое расстояние → граница кластера;

светлый цвет (белый/светло‑серый) → малое расстояние → внутри плотного кластера.

Размер: совпадает с размером карты Кохонена (m×n).

Число карт: всегда 1.
