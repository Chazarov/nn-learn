# topologic distance

Расстояние между позициями нейронов на **2D-карте** размера $m \times n$.

Нейроны индексируются линейно: $idx = r \cdot \text{cols} + c$, где
$r \in \{0, \dots, m-1\}$, $c \in \{0, \dots, n-1\}$.
Координаты нейрона-победителя получаются как $(r_c, c_c) = \mathrm{divmod}(c,\ \text{cols})$.

Значение `cols` задаётся при создании калькулятора (см. конструктор
`ITopologicCalculator(cols)`). Параметр `neurons_count`, передаваемый в `perform`,
должен быть кратен `cols` — `rows` вычисляется как `neurons_count // cols`.

## Евклидово топологическое расстояние

**Формула:**
$$d_i = \sqrt{(r_i - r_c)^2 + (c_i - c_c)^2}$$

**Где:**
- $d_i$ — топологическое расстояние до нейрона $i$
- $(r_i, c_i)$ — координаты нейрона $i$ на решётке
- $(r_c, c_c)$ — координаты нейрона-победителя

Расстояние возвращается **без** возведения в квадрат: квадрат уже включён
в гауссовскую функцию соседства ($e^{-d^2/2\sigma^2}$).

## Манхэттенское топологическое расстояние

**Формула:**
$$d_i = |r_i - r_c| + |c_i - c_c|$$

**Где:**
- $d_i$ — топологическое расстояние до нейрона $i$
- $(r_i, c_i)$ — координаты нейрона $i$ на решётке
- $(r_c, c_c)$ — координаты нейрона-победителя
