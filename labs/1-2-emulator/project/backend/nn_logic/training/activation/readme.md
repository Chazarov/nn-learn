# Функции активации

Функция softmax задаётся экспонентой с нормировкой, а её производная выражается через якобиан (диагональная матрица минус внешнее произведение вектора на себя).

## Softmax: скалярная форма

$$
s_i = \frac{\exp(z_i)}{\sum_{j=1}^{K} \exp(z_j)}, \quad i = 1, \dots, K
$$

Где:
- $s_i$ — значение softmax для $i$-го элемента
- $z_i$ — логит $i$-го элемента
- $\sum_{j=1}^{K} \exp(z_j)$ — общий нормирующий знаменатель
- $K$ — число элементов (классов)

### Производная softmax: векторная (матрица Якоби)
Вычисляется через матрицу Якоби
$$
J = \frac{\partial \mathbf{s}}{\partial \mathbf{z}} = \operatorname{diag}(\mathbf{s}) - \mathbf{s}\mathbf{s}^\top
$$

Где:
- $J \in \mathbb{R}^{K \times K}$ — матрица Якоби $\partial s_i / \partial z_j$
- $\operatorname{diag}(\mathbf{s})$ — диагональная матрица с элементами $s_1, \dots, s_K$ на диагонали
- $\mathbf{s}\mathbf{s}^\top$ — внешнее произведение вектора $\mathbf{s}$ на себя
- $\mathbf{s}$ — Выходной вектор слоя после активации
- $\mathbf{z}$ — вектор логитов

### Производная softmax: скалярная форма

$$
\frac{\partial s_i}{\partial z_j} =
\begin{cases}
s_i(1 - s_i), & \text{если } i = j, \\
-\,s_i s_j, & \text{если } i \neq j
\end{cases}
$$

Где:
- $\frac{\partial s_i}{\partial z_j}$ — частная производная выхода $i$ по входу $j$
- $s_i$ — значение softmax для $i$-й компоненты
- $s_j$ — значение softmax для $j$-й компоненты
- $i, j$ — индексы компонент векторов $\mathbf{s}$ и $\mathbf{z}$

### Матрица Якоби
Формула получения матрицы якоби из вектора s:
$$
J  = \operatorname{diag}(\mathbf{s}) - \mathbf{s}\mathbf{s}^\top
$$
Матрица Якоби получается из диагональной матрицы ( ${diag}(\mathbf{s})$ ). 
Диагональная матрица строится на основе одного вектора. Этот вектор становится диагональю матрицы , а все остальные ее элементы остаются равными 0. 

исходный вектор: s =  [1, 2, 3] -> Диагональная матрица:
diag(s) = 
| | | |
|---|---|---|
| 1    | 0    | 0    |
| 0    | 2    | 0    |
| 0    | 0   | 3    |

Далее чтобы получить матрицу Якоби из нее необходимо вычесть  внешнее произведение (outer product) вектора s: $$\mathbf{s}\mathbf{s}^\top = $$

| |  |   |  |  |  | |
|----|----|----|------|----|----|----|
| 1×1 | 1×2 | 1×3 |   | 1 | 2 | 3 |
| 2×1 | 2×2 | 2×3 | = | 2 | 4 | 6 |
| 3×1 | 3×2 | 3×3 |   | 3 | 6 | 9 |
